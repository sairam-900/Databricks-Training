# Project Overview

This project is designed to practice SQL Joins and Set Operators using multiple relational tables. It covers beginner to advanced SQL concepts commonly asked in technical interviews and placement coding rounds.

The project includes real-world datasets for employees, departments, projects, customers, and orders to understand relationships between tables and perform complex SQL operations.

---

# Dataset

The project contains the following tables:

* Departments
* Employees
* Projects
* EmployeeProjects
* CurrentEmployees
* FormerEmployees
* DomesticCustomers
* InternationalCustomers
* OnlineOrders
* StoreOrders
